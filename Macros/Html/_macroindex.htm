Urban-Greater DC Macro Library

Macro Description
Annchg Calculate annual percentage change between two numbers, <start> and <end>.
Archive_data_sets.sas Autocall macro to archive DCData data sets. Note: Overwrite=N (default) prevents replacement of older files in a previously existing archive.
Concat_lib Define a concatenated V8 & V6 SAS library
Concat_lr_lib Define concatenated local/remote/confidential SAS library. Note: Local and remote library base names must be shortened to 6 chars to allow _r and _l suffixes. Can't define two different libraries that have same first 6 chars in name.
ConvertLibname Return local library name.
DCData_lib Define NeighborhoodInfo DC SAS libraries for local and remote sessions. Sets macro and format search paths.
DC_update_meta_file Update file entry in metadata system.
DC_update_meta_library Update library entry in metadata system.
Finalize_data_set.sas Autocall macro to finalize SAS data set in DCData.
GetProgLib Return the library of the currently submitted SAS program to the global macro variable given by VAR. If running in interactive mode, macro returns an empty string.
Is_remote_batch Returns 1 if current SAS session is a remote batch submit, 0 if local batch or interactive.
Label_var_years Label a series of variables across multiple years.
mif_select.sas Returns <a> if <condition> is true, <b> if false.
Pctchg Calculate percentage change between two numbers, <start> and <end>.
Pct_calc Calculate percentages or rates across multiple years.
SAS_compatibility Set backward compatibility system options for later versions of SAS.
Set_mrecall Set MRECALL option based on value of _PROGRAM macro var.
