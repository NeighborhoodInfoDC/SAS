NeighborhoodInfo DC Macro Library

Macro Description
Annchg Calculate annual percentage change between two numbers, <start> and <end>.
Block00_to_anc02 Convert Census block IDs (2000) to DC ANCs (2002).
Block00_to_anc12 Convert Census block IDs (2000) to DC ANCs (2012).
Block00_to_bg00 Convert Census block IDs (2000) to block groups (2000).
Block00_to_casey_nbr2003 Convert Census block IDs (2000) to Casey target area neighborhood (2003).
Block00_to_casey_ta2003 Convert Census block IDs (2000) to Casey target area (2003).
Block00_to_city Convert Census block IDs (2000) to Washington, D.C..
Block00_to_cluster00 Convert Census block IDs (2000) to DC Neighborhood Clusters (2000).
Block00_to_cluster_tr00 Convert Census block IDs (2000) to DC Neighborhood Clusters (tract-based, 2000).
Block00_to_eor Convert Census block IDs (2000) to East of the Anacostia River.
Block00_to_psa04 Convert Census block IDs (2000) to DC PSAs (2004).
Block00_to_psa12 Convert Census block IDs (2000) to DC PSAs (2012).
Block00_to_tr00 Convert Census block IDs (2000) to Census tracts (2000).
Block00_to_tr10 Convert Census block IDs (2000) to Census tracts (2010).
Block00_to_vp12 Convert Census block IDs (2000) to Voting Precincts (2012).
Block00_to_ward02 Convert Census block IDs (2000) to DC wards (2002).
Block00_to_ward12 Convert Census block IDs (2000) to DC wards (2012).
Block00_to_zip Convert Census block IDs (2000) to DC ZIP code.
Block10_to_anc02 Convert Census block IDs (2010) to DC ANCs (2002).
Block10_to_anc12 Convert Census block IDs (2010) to DC ANCs (2012).
Block10_to_bg10 Convert Census block IDs (2010) to block groups (2010).
Block10_to_city Convert Census block IDs (2010) to Washington, D.C..
Block10_to_cluster00 Convert Census block IDs (2010) to DC Neighborhood Clusters (2000).
Block10_to_cluster_tr00 Convert Census block IDs (2010) to DC Neighborhood Clusters (tract-based, 2000).
Block10_to_eor Convert Census block IDs (2010) to East of the Anacostia River.
Block10_to_psa04 Convert Census block IDs (2010) to DC PSAs (2004).
Block10_to_psa12 Convert Census block IDs (2010) to DC PSAs (2012).
Block10_to_tr00 Convert Census block IDs (2010) to Census tracts (2000).
Block10_to_tr10 Convert Census block IDs (2010) to Census tracts (2010).
Block10_to_vp12 Convert Census block IDs (2010) to Voting Precincts (2012).
Block10_to_ward02 Convert Census block IDs (2010) to DC wards (2002).
Block10_to_ward12 Convert Census block IDs (2010) to DC wards (2012).
Block10_to_zip Convert Census block IDs (2010) to DC ZIP code.
Concat_lib Define a concatenated V8 & V6 SAS library
Concat_lr_lib Define concatenated local/remote/confidential SAS library. Note: Local and remote library base names must be shortened to 6 chars to allow _r and _l suffixes. Can't define two different libraries that have same first 6 chars in name.
ConvertLibname Return local library name.
Create_all_summary_from_tracts Create all summary level files from a tract level summary file.
Create_summary_from_tracts Create geographic summary data set from a tract summary data set for the specified geography.
DCData_lib Define NeighborhoodInfo DC SAS libraries for local and remote sessions. Sets macro and format search paths.
DC_update_meta_file Update file entry in metadata system.
DC_update_meta_library Update library entry in metadata system.
Fedtractno Create FEDTRACTNO variable for matching census tract data to OCTO tract shape file.
Fedtractno_geo2000 Convert Fedtractno in OCTO tract shapefile to standard Geo2000 tract ID.
GeoBlk2000_Octo Convert NeighborhoodInfo DC standard var GeoBlk2000 to OCTO census block ID (cjrTractBl).
GetProgLib Return the library of the currently submitted SAS program to the global macro variable given by VAR. If running in interactive mode, macro returns an empty string.
Is_remote_batch Returns 1 if current SAS session is a remote batch submit, 0 if local batch or interactive.
Label_var_years Label a series of variables across multiple years.
Metro03 Add 2003 metropolitan/micropolitan statistical area code (METRO03). Notes: Must have Cen2000 format library in FMTSEARCH.
Octo_Anc2002 Convert OCTO ANC ID to NeighborhoodInfo DC standard var Anc2002.
Octo_Anc2012 Convert OCTO ANC ID to NeighborhoodInfo DC standard var Anc2012.
Octo_Cluster2000 Convert OCTO neighborhood cluster ID to NeighborhoodInfo DC standard var Cluster2000.
Octo_GeoBlk2000 Convert OCTO census block ID to NeighborhoodInfo DC standard var GeoBlk2000.
Octo_GeoBlk2010 Convert OCTO census block ID to NeighborhoodInfo DC standard var GeoBlk2010.
Octo_Psa2004 Convert OCTO PSA ID to NeighborhoodInfo DC standard var Psa2004.
Octo_Psa2012 Convert OCTO PSA ID to NeighborhoodInfo DC standard var Psa2012.
Octo_Quad Convert OCTO quadrant ID to NeighborhoodInfo DC standard var Quad.
Octo_VoterPre2012 Convert OCTO Name var to NeighborhoodInfo DC standard var VoterPre2012.
Octo_ward2002 Convert OCTO ward id to NeighborhoodInfo DC standard var Ward2002.
Octo_Ward2012 Convert OCTO ward id to NeighborhoodInfo DC standard var Ward2012.
Octo_ZIP Convert OCTO ZIP Code to NeighborhoodInfo DC standard var ZIP.
Pctchg Calculate percentage change between two numbers, <start> and <end>.
Pct_calc Calculate percentages or rates across multiple years.
SAS_compatibility Set backward compatibility system options for later versions of SAS.
Set_mrecall Set MRECALL option based on value of _PROGRAM macro var.
Tr00_to_city Convert tract ID to city ID.
Tr00_to_cnb03 Convert tract ID to Casey Target Neighborhoods ID.
Tr00_to_cta03 Convert tract ID to Casey Target Area ID.
Tr00_to_eor Convert tract ID to EOR ID.
Tr10_to_eor Convert 2010 tract ID to EOR ID.
